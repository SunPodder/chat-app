# Chat App
A chat app made with React, Express, SocketIO, PostgreSQL

# Run
Assuming you've bun installed.

Setup:
```bash
bun install
docker-compose pull
```

Run:
```bash
./run.sh
```

- It'll up a postgresql and redis server
- Start the backend API server
- Build the frontend and start a vite preview server
